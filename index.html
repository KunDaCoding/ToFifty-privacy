Privacy Policy
Last updated: 2026-05-11
01
Data Collection
本遊戲收集以下資料，用於帳號識別與排行榜功能的正常運作：
  • Apple Game Center 玩家識別碼（Game Center Player ID）— 用於唯一識別玩家身份，防止排行榜作弊
  • 玩家暱稱（Nickname）— 玩家自行輸入
  • 所在地區（Region）— 玩家自行選擇
  • 性別（Gender）— 玩家自行選擇
  • 遊戲成績（Game Score）— 最佳成績與歷史紀錄
  • 裝置產生的匿名識別碼（Anonymous Device UID）
  • 購買狀態（Purchase Status）— 是否已購買遊戲內付費功能
02
Purpose of Use
所有收集的資料僅用於以下目的：
  • 顯示全球與地區排行榜
  • 透過 Apple Game Center 驗證玩家身份，確保每位玩家在排行榜上擁有唯一位置
  • 在玩家更換裝置或重新安裝時還原遊戲進度
  • 記錄付費功能解鎖狀態
我們不會將您的資料出售、租借或提供給任何第三方，也不會用於廣告或行銷目的。
03
Third-Party Services
本遊戲使用以下第三方服務處理部分資料：
  • Apple Game Center — 用於玩家身份驗證。適用 Apple 隱私政策：apple.com/legal/privacy
  • Supabase — 用於雲端儲存玩家資料（暱稱、地區、成績等）。適用 Supabase 隱私政策：supabase.com/privacy
  • Apple In-App Purchase（StoreKit） — 用於處理遊戲內購買交易。付款資訊由 Apple 直接處理，本遊戲不會儲存任何付款資訊。
04
TrueDepth API & Face Data
本遊戲使用 Godot Engine（版本 4.6.2）及 GodotApplePlugins 框架作為開發基礎。這些第三方工具的底層二進位檔案中包含對 Apple TrueDepth API 的靜態參照，此為引擎與插件在編譯時自動引入的間接依賴，並非本遊戲主動使用。

茲聲明如下：
  • 本遊戲不存取、不收集、不處理、不儲存任何臉部資料或深度感測器資料
  • 本遊戲不包含任何臉部辨識、擴增實境（AR）或 Face ID 相關功能
  • TrueDepth API 從未被本遊戲的程式碼呼叫或啟用
  • 任何臉部相關資料均不會與第三方共享，因為本遊戲根本不收集此類資料
05
Data Storage & Security
玩家資料儲存於 Supabase 提供的雲端伺服器。我們採取合理的技術措施保護您的資料安全。本遊戲不會收集任何敏感個人資訊，如電子郵件地址、電話號碼或位置資訊。
06
Data Deletion
您可以隨時在遊戲內的設定頁面中刪除您的帳號。刪除後，您的所有資料將從我們的伺服器上永久移除，且無法復原。Apple Game Center 帳號不受影響，僅刪除本遊戲相關的資料。
07
Children's Privacy
本遊戲適合所有年齡層遊玩。我們不會在知情的情況下向 13 歲以下兒童收集超出遊戲功能所需的個人資料。若您認為您的孩子提供了不當資料，請透過下方聯絡方式告知我們，我們將盡快處理。
